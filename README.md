# ItCourseTestApp

Тестовое задание для разработки приложения курсов под Android.

## Стек технологий

Проект разработан с использованием стека разработки под Android, рекомендованного в ТЗ:

- **Язык**: Kotlin
- **Архитектура**: MVVM + Clean Architecture
- **Многомодульность**: проект разбит на 4 модуля (`app`, `domain`, `data`, `presentation`)
- **Асинхронность**: Kotlin Coroutines + Flow (StateFlow, SharedFlow)
- **Сеть**: Retrofit + Gson
- **Локальная БД**: Room 
- **Dependency Injection**: Koin
- **UI**: XML, ViewBinding, AdapterDelegates
- **UI-эффекты**: Facebook Shimmer

## Реализованный функционал (согласно ТЗ)

1. **Экран "Вход"**
   - Валидация Email по строгой маске `текст@текст.текст` (запрещена кириллица).
   - Валидация пароля (любые символы).
   - Интерактивная кнопка входа, которая разблокируется только при валидных данных.
   - Кнопки ВК и Одноклассников открывают системный браузер по соответствующим ссылкам.

2. **Главный экран (Курсы)**
   - Получение данных курсов по API.
   - Красивая UI-карточка, 1 в 1 соответствующая Figma-макетам.
   - Описание курса ограничено двумя строками с "т...".
   - Сортировка списка по дате добавления (по убыванию `publishDate`).
   - Добавление и удаление курса из "Избранного" по клику на флажок (флажок закрашивается зеленым). 
   - Данные об избранном сохраняются локально в Room DB!

3. **Избранное**
   - Вывод только тех курсов, у которых установлен флаг `hasLike == true`.
   - При снятии лайка курс плавно анимировано пропадает из списка.

4. **Меню (Bottom Navigation)**
   - Главная
   - Избранное
   - Аккаунт (экран-заглушка)

## Как запустить

1. Склонируйте репозиторий.
2. Откройте в Android Studio (проверено на AGP 8+ и Kotlin 2+).
3. Соберите проект (`Sync Project with Gradle Files`).
4. Нажмите `Run app` для установки на эмулятор или реальное устройство.

**Примечание:** Для работы KSP с последними версиями Kotlin в проекте используется экспериментальное рекомендованное решение `android.disallowKotlinSourceSets=false` в файле `gradle.properties`.
